# Credit Ratings vs Default Outcomes | Chi-Square Analysis

Do credit ratings actually predict bond defaults?

I applied a Chi-Square Test of Independence to 240 Indian corporate bonds 
(CRISIL / ICRA ratings) to test whether credit rating and default outcome 
are statistically associated.

## Key Finding
χ² = 66.81 against a critical value of 21.03 (df = 12, α = 0.05)- strongly 
rejects the null hypothesis. Credit ratings carry meaningful, statistically 
significant information about default risk in Indian corporate bond markets.

## Methodology
- Dataset: 240 Indian corporate bonds (CRISIL / ICRA rated)
- Test: Chi-Square Test of Independence
- Structure: 4×5 contingency table (rating categories vs default outcomes)
- Significance level: α = 0.05

## Implications
Rating categories are not arbitrary classifications — they capture genuine 
variation in default probability. This supports their use as a risk 
stratification tool in fixed income research and credit portfolio management.
